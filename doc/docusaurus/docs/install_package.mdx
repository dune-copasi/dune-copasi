Debian 11 users who want to use DuneCopasi with its [Command-Line-Interface](use_cli) can install it through the package manager:

### Install

<Tabs
  groupId="package-manager"
  defaultValue="apt"
  values={[
      {label: 'Debian 11 (apt)', value: 'apt', },
    ]
  }>

  <TabItem value="apt">

```bash
curl -fsSL https://gitlab.dune-project.org/copasi/dune-copasi/-/package_files/3908/download -o dune-copasi-runtime.deb
apt install ./dune-copasi-runtime.deb
```

  </TabItem>
</Tabs>

Once installed, the program `dune-copasi` will be available on the command line:

<DuneCopasiHelp/>

### Uninstall

To remove the package, remove the `dune-copasi-runtime` with the package manager:

<Tabs
  groupId="package-manager"
  defaultValue="apt"
  values={[
      {label: 'Debian 11 (apt)', value: 'apt', },
    ]
  }>

  <TabItem value="apt">

```bash
apt remove dune-copasi-runtime
```

  </TabItem>
</Tabs>
